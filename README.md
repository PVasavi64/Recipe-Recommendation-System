# Recipe-Recommendation-System
## Project Overview

Mom's Kitchen is a simple and interactive recipe website built using HTML, CSS, and JavaScript. The website allows users to search for recipes and view ingredients and cooking instructions through a clean and user-friendly interface.

## Features

* Recipe search functionality
* Attractive and responsive design
* Easy-to-read recipe instructions
* Recipe ingredient lists
* Interactive user interface
* Beginner-friendly project structure

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

```text
MomsKitchen-RecipeFinder/
│

└── README.md
```

## How to Run the Project

1. Download or clone the repository.
2. Make sure all files are in the same folder.
3. Open `index.html` in any modern web browser.
4. Use the search bar to search for available recipes.
